# synapse-ai-multi-agent

A lightweight 2-Agent Research and Summarization system built with Python, Pydantic, and Gemini API.

## Features
- **Research Agent**: Utilizes mock search tool to fetch information and structured key findings.
- **Summarizer Agent**: Synthesizes structured findings into user-facing bullet points and next steps.
- **Structured Output**: Strict schema enforcement using Pydantic.
- **Comparison Engine**: Run single-agent vs 2-agent comparison.
